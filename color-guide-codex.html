02 漸層方式

Radial + Canvas
Base Radial Gradient
radial-gradient(circle at center, #9c005d 0%, #1a0512 50%, #0d0209 100%)
Overlay Radial Gradient
radial-gradient(circle at center, rgba(156, 0, 93, 0.3) 0%, rgba(156, 0, 93, 0.6) 50%, rgba(13, 2, 9, 0.9) 100%)
Canvas Line Gradient (Stroke)
linear-gradient(0deg, rgba(189,161,108,α), rgba(212,184,138,α)) with per-line opacity

03 Render 方式

Stacked Layers
Hero 疊三層：底層為 body/hero 的放射漸層背景，其上覆蓋一層較高對比的放射 overlay，最上層是 canvas 畫的波浪線條。 Canvas 線條每條帶有線性漸層，並依 y 位置調整透明度，產生中心較亮、上下較暗的視覺。
Layer 1
Base Radial
hero 或 body 背景
Layer 2
Overlay Radial
提升中心亮度、壓暗邊緣
Layer 3
Canvas Lines
金色線條細節與動態

Migration Checklist

套用到 index.html 時需要修改的地方。

原本 Class 改為 出現位置
text-gray-300 style="color: rgba(255,255,255,0.85)" About 段落、Benefits 段落、Schedule 課程項目、Registration 資訊、Hero 主辦單位
text-gray-400 style="color: rgba(189,161,108,0.75)" Audience card 段落、Feature 小卡段落、Schedule 上午/下午 label、Hero Date/Venue label、Registration 行前準備說明
text-gray-500 style="color: rgba(189,161,108,0.55)" Footer copyright
Feature 小卡 text-sm 改為 text-sm + text-body color Feature 三欄小卡描述文字（保持 text-sm 但顏色從 gray-400 改為 white/85）

Tailwind Custom Class 建議

如要避免大量 inline style，可在 <style> 中加入以下 utility class：

/* Text Color Utilities — Gold & White Only */
.text-body      { color: rgba(255, 255, 255, 0.85); }
.text-label     { color: rgba(189, 161, 108, 0.75); }
.text-caption   { color: rgba(189, 161, 108, 0.55); }

加入後可直接用 class 取代 inline style，例如：text-gray-300 → text-body
